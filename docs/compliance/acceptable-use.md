# Acceptable Use Policy

**Owner:** CTO

All staff and contractors with access to the platform must:

- Use individually-assigned credentials only. **No shared accounts.**
- Enrol MFA when their role requires it; never share TOTP secrets.
- Access customer data only when needed for a legitimate work task.
- Never export customer data to personal devices, email, or unsanctioned cloud storage.
- Report suspected security incidents to security@busacta.com within 1 hour of detection.
- Respect the principle of least privilege — request elevated access only when justified.

Violations are grounds for access revocation and, where applicable, termination.
